Class CentralProcessingThread

  • java.lang.Object
    • java.lang.Thread
      • com.waoss.enesys.cpu.CentralProcessingThread
  • All Implemented Interfaces:
    java.lang.Runnable

    public class CentralProcessingThread
extends java.lang.Thread

    A thread in which the central processing of the CPU occurs. The CPU uses the CentralProcessor.start() method to start processing. Processing starts a loop from the program counter and the thread reads every byte and finds the corresponding instruction and then uses the CentralProcessor.process(Instruction) function to process the instruction

    See Also:
    CentralProcessor, CentralProcessor.process(Instruction), CentralProcessor.start(), Console
