Package com.waoss.enesys.cpu.registers Description

This package is practically the register file of the CPU.The 6502 had only three general purpose registers :
  • The Accumulator AccumalativeRegister
  • The X register XRegister
  • The Y register YRegister

Except this, there was the ProgramCounter and the StackPointer.All these are different implementations of Register.
